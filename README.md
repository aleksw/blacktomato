Black Tomato
============

No-frills [Pomodoro](https://en.wikipedia.org/wiki/Pomodoro_Technique) timer for the system tray. Requires Qt5.

Compile
-------

```
cd blacktomato
mkdir build
cd build
qmake ..
make
```
