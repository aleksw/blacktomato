Black Tomato
============

No-frills [Pomodoro](https://en.wikipedia.org/wiki/Pomodoro_Technique) timer for the system tray. Displays notifications between count modes.
Requires Qt5.

Compile
-------

```
cd blacktomato
mkdir build
cd build
qmake ..
make
```
*Compiled and tested on Linux (KDE) and Windows 7*
